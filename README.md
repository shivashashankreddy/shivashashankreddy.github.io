# Shiva Shashank Chevella Portfolio

This repository contains Shiva Shashank Chevella's GitHub Pages portfolio website.

The site is built with static HTML, CSS, and minimal JavaScript. It is designed to run directly on GitHub Pages without React, Node.js, npm, or external build tools.

## Content

Portfolio content is based on Shiva Shashank Chevella's professional resume and focuses on senior GenAI engineering, Agentic AI, RAG, LLMOps, cloud-native AI platforms, and regulated enterprise delivery.

## Files

- `index.html` - Main portfolio page
- `styles.css` - Dark theme and responsive layout styles
- `script.js` - Mobile navigation and footer year behavior
- `README.md` - Project documentation

## Run Locally

Open `index.html` directly in a browser.

## Deploy

Push the repository to the `main` branch and enable GitHub Pages for the repository. GitHub Pages can serve the site directly from the repository root.
